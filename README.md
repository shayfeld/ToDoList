## Tech Stack

**Client:** React, Redux, Bootstrap

**Server:** NodeJS, Express 

**Database:** MongoDB
## Run Locally

Clone the project

```bash
  git clone https://github.com/shayfeld/ToDoList.git
```

Go to the project directory

```bash
  cd TodoList
```

Install dependencies

```bash
  npm install
```

Start the client & server

```bash
  npm run start-dev
```

Open in browser

```bash
  http://localhost:8080/
```
